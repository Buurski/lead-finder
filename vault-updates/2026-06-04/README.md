# Vault-opdateringer — 2026-06-04 (Del 3)

Disse noter er skrevet af command-center'et, men ligger HER (ikke i `KnowledgeOS/`)
med vilje: den lokale `KnowledgeOS/`-mappe er kun et seed, og jeg vil ikke skygge
for din rigtige remote-vault (`Buurski/KnowledgeOS`) som appen læser live fra.

## Sådan tager du dem i brug
1. Kopiér filerne ind i din Obsidian-vault på de tilsvarende stier:
   - `wiki/os/hermes-detaljeret.md` → ny note
   - `wiki/os/strategiske-ideer-park.md` → ny note
   - `wiki/proces/outreach-systemet.md` → ny/opdateret note
   - `*.APPEND.md` → indsæt indholdet i bunden af den eksisterende note
2. Tilføj links i din `index.md` (forslag i `index-links.md`).
3. `git push` på `Buurski/KnowledgeOS` (kun det repo — ikke command-center-v3).

Jeg pusher dem ikke selv: det er et separat repo, og en push er udadvendt.
